Monitor de Hardware

Uma aplicação simples para coletar, armazenar e visualizar métricas de dispositivos em tempo real.

Sobre o projeto

Este sistema foi desenvolvido para monitorar diferentes dispositivos em rede, coletando informações como temperatura, uso de recursos e conectividade. Os dados são enviados por agentes distribuídos e centralizados em uma API, permitindo visualização e análise de forma simples.

A ideia é manter tudo leve, modular e fácil de expandir conforme a necessidade.
